Featurejs.com

How to run locally

  1. You need CodeKit
  2. Add feature.js as an existing project to CodeKit
  3. Done!

Alternatively, if you just want to see the site locally on OS X, you can run python -m SimpleHTTPServer 8080 in the folder and then point your browser to http://localhost:8080
